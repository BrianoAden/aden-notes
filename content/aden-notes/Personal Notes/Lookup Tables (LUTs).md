Lookup Tables are digital circuits that use truth tables to configure logical outputs.  

The simplest FPGAs that came out in the 80s had 3-input LUTs. We could represent any 3-input boolean algebra equation using these LUTs. 

Basically, a LUT contains small RAM memory blocks to store the outputs of a truth table. Based on the inputs into the LUT, it will output a specific bit from memory.
