**What is a computer?**
- Every computer has key five components:
	  - The processor gets instructions and data from **memory**
	  - **Input** writes data to memory
	  - **Output** reads data from memory
	  - **Control** sends the signals that determine the operations of the computer
	  - **Datapath** contains the memory, input, and output
- We will often refer to **Control** and **Datapath** together as the entire **Processor**

Notes:

Memory is there to load data into datapath, and store data from datapath. Control tells the processor which sort of operation we're performing based on input instructions.
