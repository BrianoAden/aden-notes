## What is a Pragma?

A pragma is a special compiler directive, also known as a pragma, that provides extra information to guide the HLS tool in optimizing C/C++ code into hardware

## Vitis HLS Parameters

Vitis HLS interfaces provided coordination between ARM core PS and FPGA PL

AXI is standard for communication

3 flavors, Lite, Memory, Stream

Transfers are 32 unsigned values
